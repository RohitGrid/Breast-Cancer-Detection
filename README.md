<h1 align="center">🧬 Breast Cancer Classification using Machine Learning</h1>

<p align="center">
  A machine learning approach for early detection of breast cancer by classifying tumors as benign or malignant
</p>

<hr/>

<h2>🚀 Project Overview</h2>
<p>
This project develops a machine learning model trained on the Breast Cancer Wisconsin dataset to classify tumors as <b>benign</b> or <b>malignant</b>. 
It demonstrates the complete ML pipeline including data preprocessing, visualization, model training, and prediction.
</p>

<hr/>

<h2>✨ Features</h2>
<ul>
  <li>🧠 <b>Logistic Regression Model</b> for binary classification</li>
  <li>📊 <b>Exploratory Data Analysis (EDA)</b> with visualizations</li>
  <li>📉 <b>Pairplot & Scatterplot</b> for feature relationships</li>
  <li>📈 <b>Confusion Matrix & Metrics</b> for evaluation</li>
  <li>⚡ <b>Prediction System</b> for new input data</li>
</ul>

<hr/>

<h2>📊 Dataset</h2>
<p>
The model uses the <b>Breast Cancer Wisconsin Dataset</b>, which contains features computed from digitized images of breast mass samples.
</p>

<ul>
  <li>🔢 <b>Total Features:</b> 30 numerical features</li>
  <li>🎯 <b>Target Classes:</b>
    <ul>
      <li>0 → Malignant (Cancerous)</li>
      <li>1 → Benign (Non-cancerous)</li>
    </ul>
  </li>
</ul>

<hr/>

<h2>⚙️ Workflow</h2>
<ul>
  <li>Data loading and preprocessing</li>
  <li>Exploratory Data Analysis (heatmap, pairplot, scatterplot)</li>
  <li>Feature scaling using StandardScaler</li>
  <li>Model training using Logistic Regression</li>
  <li>Evaluation using accuracy and classification metrics</li>
  <li>Prediction on new input data</li>
</ul>
